# Morgan Desktop Voice Assistant
*I creat this assistant to increase the productivity of my daily life*

# Requirements to run Morgan
-Python 3.9
-pyttsx3 module
-wikipedia module
-webbrowser module
-os module
-smtplib module
-psutil module
-pyautogui
-screen_brightness_control
-ctypes module
-keyboard module
